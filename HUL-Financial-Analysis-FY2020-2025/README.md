# Hindustan Unilever Financial Analysis (FY2020–FY2025)

## Overview
This project presents a financial analysis of Hindustan Unilever Limited (HUL) from FY2020 to FY2025.  
The objective of this project is to analyze the company's financial performance using key financial ratios, revenue trends, and interactive dashboards.

The analysis helps in understanding profitability, efficiency, and financial stability over multiple financial years.

---

## Tools Used
- Microsoft Excel
- Microsoft Power BI

---

## Key Analysis Performed
- Revenue trend analysis
- Total assets and liabilities analysis
- Debt-to-equity ratio
- Equity ratio
- Debt ratio
- Asset turnover ratio
- Financial performance comparison across multiple years

---

## Dashboard Features
- Interactive Power BI dashboard
- Year-wise financial performance analysis
- Financial ratios visualization
- Trend analysis of key financial metrics

---

## Files Included
- `HUL-Financial-Dashboard.pbix` – Power BI dashboard file
- `HUL-Financial-Analysis.pdf` – Project report
- `dataset.xlsx` – Financial dataset used for analysis
- `dashboard-screenshot.png` – Preview of the dashboard

---

## Insights
- The analysis highlights trends in revenue growth and financial structure.
- Ratio analysis helps evaluate company efficiency and leverage.
- Visualization enables easier interpretation of financial data.
